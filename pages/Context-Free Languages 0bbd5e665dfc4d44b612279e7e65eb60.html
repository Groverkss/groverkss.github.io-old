Context-Free Grammars

A grammar consists of a collection of substitution rules.

Each rule is a variable → string.

The string consists of other variables and terminals.

The variable symbols are usually represented by capital letters.

The variable on left-hand side of the topmost rule is by convention the start variable.

Ambiguity

The same string can have different parse trees. This is unacceptable for some applications such as programming languages where we need a unique interpretation.

Note that we say different parse trees and not different derivations. Derivations can differ merely in structure. To make it more concrete, we define leftmost derivation.

A derivation of a string www﻿ in a grammar GGG﻿ is a leftmost derivation if at every step the leftmost remaining variable is the one replaced.

For some ambiguous languages we can find an unambiguous grammar that generates the same language.

Languages which can only be generated by ambiguous grammars are called inherently ambiguous.

Chomsky Normal Form

Theorem: Any context-free language is generated by a context-free grammar in Chomsky normal form.

Pushdown Automata

Like NFAs but have an extra component which provides additional memory, called stack.

Pushdown automata are equivalent in power to context-free grammars.

The figure on right represents a finite automaton. The control represents states and transition function. The tape contains input and arrow represents input head pointing towards next symbol to be read.
