Centroid Decomposition

What is Centroid and How to Find it?

Centroid is a node such that if you remove it, the graph breaks into components of size at max n2\frac{n}{2}2n​﻿, where nnn﻿ is the number of nodes in graph.

How to find a centroid?

Look at subtrees of current node, if all subtrees have ≤n2\le \frac{n}{2}≤2n​﻿ elements, then this node is the centroid.

Otherwise, there will be only one node which has >n2\gt \frac{n}{2}>2n​﻿ nodes (obviously).

We move to the node having more than >n2\gt \frac{n}{2}>2n​﻿ nodes in its subtree, and repeat this process.

This can be done in O(n)O(n)O(n)﻿.

Theorem: Centroid always exists for a Tree.

Starting from a node XXX﻿ (current root), if we move to a node which is a child of this node YYY﻿, we do not need to check if XXX﻿ w.r.t YYY﻿ has >n2\gt \frac{n}{2}>2n​﻿ nodes since the subtree of node YYY﻿ had >n2\gt \frac{n}{2}>2n​﻿, which implies the set all other nodes (subtree of XXX﻿ w.r.t. YYY﻿) has cardinality <n2\lt \frac{n}{2}<2n​﻿.

Centroid Decomposition is converting the tree into a tree of centroids.
