Turing Machines

Similar to a finite automaton but with an unlimited and unrestricted memory.

Much more accurate model of a general purpose computer.

Can do everything a real computer can do.

A language is called Turing-recognizable if some Turing machine recognizes it. (Also called recursively enumerable language).
A language is called Turing-decidable if some Turing machine decides it. (Also called recursive language).

Difference between recognizable and decidable:

Deciders halt on every valid input.

Recognizers halt on every valid input which is in the language but may or may not halt on an input which is not part of the language.

Every decidable language is Turing-recognizable but the converse is not true.

Variants of Turing Machines

The original model and its reasonable variants all have the same power, i.e. they can recognize the same class of languages.

This invariance to certain changes in the definition is called robustness.

Turing machines have an astonishing degree of robustness.

Multitape Turing Machines

Variation → Instead of one tape, several tapes are there each with its own head for reading and writing.

Theorem: Every multitape Turing machine has an equivalent single-tape Turing machine.

Proof Idea:

Interleave the tapes into a single tape.

Corollary: A language is Turing-recognizable if and only if some multitape Turing machine recognizes it.

Non-Deterministic Turing Machines

At a point, the machine may proceed according to several possibilities.

Theorem: Every nondeterministic Turing machine has an equivalent deterministic Turing machine.

Proof Idea:

Consider the computation of the turing machine over some input as a tree.

Now, the idea is to try all branches. This will be done in a BFS order as the DFS order may pick a never ending branch and miss some accepting branch.

This guarantees that the machine will visit every node in the tree until it encounters an accepting configuration.

Corollary: A language is Turing-recognizable if and only if some nondeterministic Turing machine recognizes it.
Corollary: A language is decidable if and only if some nondeterministic Turing machine decides it.
