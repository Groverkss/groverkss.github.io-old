MDPs

The environment in which a agent is places may subject the agent's actions to being nondeterministic, which means there are multiple possible successor states that can result from an action taken in some state.

Such problems which have a degree of uncertainty are known as non deterministic search problems and are modelled as Markov decision processes or MDPs.

Markov Decision Processes

A markov decision process is defined by several properties:

  1. A set of states SSS﻿
  1. A set of actions AAA﻿
  1. A start state
  1. One or more terminal states
  1. Possibly a discount factor γ\gammaγ﻿.
  1. A transition function T(s,a,s′)T(s, a, s\prime)T(s,a,s′)﻿. TTT﻿ is a function that represents the probability that an agent taking an action a∈Aa \in Aa∈A﻿ from a state s∈Ss \in Ss∈S﻿ ends up in a state s′∈Ss\prime \in Ss′∈S﻿.
  1. A reward function R(s,a,s′)R(s, a, s\prime)R(s,a,s′)﻿. The agents objective is naturally to maximize the reward possible.

Finite Horizons and Discounting

We need to place some time constraint of number of timesteps otherwise our agent can get stuck in a loop and get infinite reward.

This is prevented by adding finite horizons and/or discount factors.

Finite horizon: Add a lifetime for agents, giving them limited steps to acquire as much reward as possible.

Discount Factors: An exponential decay in the rewards over time.

Markoviness

Markovian property: Knowing the past doesn't give us any more information about the future.
