You have some data regarding size of houses vs price of house. You want to fit a function to predict the price of house as a function of size of house.

This is the simpliest possible neural network. For this network, as shown in the figure.

Lets say we have more features. Now, we want to build a function that given an instance of data, we can get a corresponding price.

What you implement is a neural network as this function. Given an input to the neural network, we get a predicted price according to it.

The inner circles are called hidden layer nodes. The neural decides itself what these nodes should mean. This is what a neural network is. A function that given an input predicts something.

Supervised Learning

TLDR: Given data, predict something.

Different type of neural networks are used for different things. For example, for the home features example we use standard neural network, for image recognition we use Convolutional Neural Networks, for sequenced data we use Recurrent Neural Networks.

Structured Data —> Data having well defined fields. For example a relational database.

Unstructured Data —> Data like audio signals, images, hand-written text. Not having well defined fields.
