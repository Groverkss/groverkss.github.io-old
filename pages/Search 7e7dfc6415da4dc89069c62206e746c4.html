Agents that plan ahead

Reflex Agents

  • Choose action based on current percept (and maybe memory(
  • May have memory or a model of the world's current state.
  • Do not consider the future consequences of their actions
  • Consider how the world IS

Can a reflex agent be rational?

Planning Agents

  • Decisions based on (hypothesized) consequences of actions.
  • Must have a model of how world evolves in response to actions
  • Must formulate a goal
  • Consider how the world WOULD BE.

Search Problems

A search problems consists of:

  • A state space
  • A successor function (with actions, costs)
  • A start state and a goal test

A solution is a sequence of actions (a plan) which transforms the start state to a goal state.

Search problems are just a modeling of a problem

What's in a State Space?

The world state includes every last detail of the environment.

A search state keeps only the details needed for planning (abstraction)

Problem: Pathing

  • States: (x, y) locations
  • Actions: NSEW
  • Successor: update locations only
  • Goal test: is (x, y) = END

Problem: Eat-All-Dots

  • States: {(x, y), dot booleans}
  • Actions: NSEW
  • Successor: Update location and possibly a dot boolean
  • Goal test: dots all false

State Space Graphs

State Space Graph: A mathematical representation of a search problem

  • Nodes are (abstracted) world configurations
  • Arcs represent successors (action results)
  • The goal test is a set of goal nodes (maybe only one)

In a state space graph, each state occurs only once

We can rarely build this full graph in memory (it's too big), but it's a useful idea.

Search Trees

  • A "what if" tree of plans and their outcomes
  • The start state is the root node
  • Children correspond to successors
  • Nodes show states, but correspond to PLANS that achieve those states
  • For most problems we actually never build the whole tree
