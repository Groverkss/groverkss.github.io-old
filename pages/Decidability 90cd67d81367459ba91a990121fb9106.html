Undecidability

Some problems are not solvable by a computer. To prove that such a problem exists, we show the following:

The problem of determining whether a Turing machine accepts a given input string is undecidable.

Theorem: ATMA_{TM}ATM​﻿ is undecidable

First, ATMA_{TM}ATM​﻿ is Turing-recognizable. Thus, this theorem also shows that recognizers are more powerful than deciders.

The following Turing machine UUU﻿ recognizes ATMA_{TM}ATM​﻿.

The Turing machine UUU﻿ is called universal Turing machine and is capable of simulating any other Turing machine from a description of that machine.
